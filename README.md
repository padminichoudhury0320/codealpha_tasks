# Task 2: Unemployment Analysis with Python

## Objective

Analyze unemployment trends in India and study the impact of COVID-19 on unemployment rates.

## Analysis Performed

* Data Cleaning
* Date Conversion
* Year-wise Analysis
* Month-wise Analysis
* State-wise Analysis
* Urban vs Rural Analysis
* Data Visualization

## Key Findings

* Unemployment increased significantly during 2020.
* April 2020 recorded the highest unemployment rate.
* Urban unemployment was higher than rural unemployment.
* States such as Tripura and Haryana showed higher unemployment rates.

## Tools Used

* Pandas
* Matplotlib

# Task 4: Sales Prediction using Python

## Objective

Predict product sales based on advertising expenditure across different marketing channels.

## Features Used

* TV Advertising
* Radio Advertising
* Newspaper Advertising

## Target Variable

* Sales

## Machine Learning Technique

* Regression
* Linear Regression

## Results

* R² Score ≈ 0.90
* MAE ≈ 1.46

## Key Findings

* TV advertising showed the strongest impact on sales.
* Radio advertising showed a moderate impact.
* Newspaper advertising showed the weakest impact.
* Increasing advertising expenditure generally resulted in higher sales.

## Outcome

Developed a predictive model to estimate future sales and provide marketing insights.
